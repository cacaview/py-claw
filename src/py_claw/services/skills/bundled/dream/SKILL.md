# Dream Skill

Explore and visualize code architecture, patterns, and designs through imaginative analysis. Generate creative interpretations of code structure to help identify improvements and innovative solutions.

## Purpose

Use creative thinking and pattern recognition to:
- Explore alternative code architectures
- Visualize code as abstract structures
- Generate innovative approaches to problems
- See connections between disparate code areas

## When to Use

Use when:
- The user says things like "imagine", "dream", "visualize", "what if"
- Exploring creative solutions to architectural problems
- Wanting to see code from a different perspective
- Brainstorming alternative approaches

## How to Use

Start a conversation with the Dream skill to explore your codebase imaginatively:

1. Share what you're working on or struggling with
2. I'll help visualize the structure and patterns
3. Together we'll explore creative alternatives
4. We can translate dreams into actionable improvements

## Dream Mode Characteristics

### Code as Architecture
- View functions as rooms in a building
- Modules as floors or wings
- Dependencies as corridors and doors
- APIs as windows connecting spaces

### Patterns as Nature
- Design patterns as plants growing in a garden
- Code smells as warning signs
- Refactorings as renovations
- Technical debt as foundations to rebuild

### Systems as Organisms
- Services as organs
- Data flow as circulation
- APIs as nervous system
- Databases as memory

## Techniques

### Free Association
Start with a code element and freely associate:
- "What does this function remind me of?"
- "What would this look like as a story?"
- "If this were a natural phenomenon, what would it be?"

### Perspective Shifting
Look at code from different viewpoints:
- What would a poet see?
- What would an architect notice?
- What would a musician hear?
- What would a painter capture?

### Metaphor Building
Create metaphors that illuminate:
- "This cache is like a museum..."
- "This API is like a translator..."
- "This database is like a library..."

## Output

The Dream skill helps you:
1. See patterns you might have missed
2. Understand complex relationships
3. Generate creative alternatives
4. Find innovative solutions

## Notes

- Dream mode is for exploration and creativity, not for getting definitive answers
- All "visions" should be grounded in practical feasibility
- Use insights from dreaming to inform actual implementation decisions
- Not all dreams need to become reality
